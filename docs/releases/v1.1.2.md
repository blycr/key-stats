## Highlights

- **Light/dark/auto theme** — theme switcher in Settings panel with light, dark, and system-auto modes. Auto mode follows the OS color scheme preference in real time.
- **Native window theming** — window chrome updates to match the selected theme via Wails runtime API.
- **CSS smooth transitions** — background, text, border, and shadow all animate with a 350ms easing when switching themes.

## Changes

- Added `theme.js` — theme manager with `applyTheme()` and `setupAutoTheme()` for `matchMedia` listener
- Converted all Tailwind color tokens from hardcoded hex to `rgb(var(--color-*) / <alpha-value>)` CSS custom properties
- Added light theme CSS variables (`[data-theme="light"]`) alongside the existing dark theme defaults
- Converted hardcoded box-shadows to `var(--shadow-*)` presets that adapt per theme
- App.svelte loads and applies saved theme on mount, sets up auto-theme listener, cleans up on destroy
- SettingsPanel.svelte applies theme immediately on save, dropdown shadows theme-aware
- Modal.svelte shadow theme-aware
- Version bump to 1.1.2

Download `key-stats.exe` and run. Data is stored at `%APPDATA%/key-stats/`.

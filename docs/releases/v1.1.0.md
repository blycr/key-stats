## Highlights

- **Left-click tray to show window** — clicking the system tray icon with the left mouse button now brings the main window to the foreground immediately
- **Native Win32 tray** — replaced `getlantern/systray` with a custom Win32 API implementation (`Shell_NotifyIcon`, `CreateWindowEx`, `TrackPopupMenu`, `GetMessage` message loop via `syscall`). Full control over tray behavior, zero third-party tray dependencies
- **AGENTS.md** — comprehensive guidance file for Claude Code / OpenCode sessions in this repo

## Changes

- Removed `getlantern/systray` and 8 transient dependencies from `go.mod` (`go.sum` shrunk by 30 lines)
- Updated CLAUDE.md and README.md — fixed build instructions, added AGENTS.md reference, corrected prerequisite versions (Go 1.25+)
- Removed obsolete `gen_ico.ps1` and `gen_ico.py` scripts (replaced by `gen_ico.go` in a previous release)
- Version bump to 1.1.0 in `wails.json`

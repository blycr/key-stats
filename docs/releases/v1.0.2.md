## Highlights

- **Date range stats** — Today button now lets you switch between Today, Yesterday, Last 7 Days, and Last 30 Days
- **Custom font configuration** — choose your preferred monospace font from Settings; reads all installed system fonts automatically
- **Monospace-only UI** — entire interface now renders in elegant monospace fonts; default is JetBrains Mono with Fira Code / Cascadia Code / Consolas fallbacks
- **Window icon fix** — taskbar thumbnail preview now correctly shows the custom app icon instead of the Windows default

## Fixes

- Fixed the unresponsive "Today" button (now opens a date-range dropdown)
- Fixed TypeScript errors in auto-generated Wails bindings (`App.js` / `App.d.ts`)
- Fixed `.syso` resource file leaking between builds (now cleaned before and after every build)
- Fixed `frontend/dist` not being rebuilt when `wails build -s` skipped the frontend step
- Switched build script from `.cmd` to `.ps1` for reliable PowerShell error handling
- Multi-size ICO generation (16×16 through 256×256) so Windows selects the optimal resolution for every context

## Build

Download `key-stats.exe` and run. Data is stored at `%APPDATA%/key-stats/`.

Real-time keyboard usage tracker for Windows. Beautiful, minimal, and stays out of your way.

## Highlights

- **Global keystroke capture** — Win32 `WH_KEYBOARD_LL` hook, works across all apps
- **Live dashboard** — today's total, top 10 keys ranking, interactive QWERTY heatmap
- **System tray** — minimize to tray, show/quit from tray menu
- **Elegant UI** — frameless window with Windows Mica backdrop, dark glassmorphism theme
- **Persistent config** — remembers window size, `.env`-based settings (theme, auto-start, portable mode)

## Bug Fixes

- Fixed button unresponsiveness caused by conflicting document-level event listeners (Svelte event delegation vs native `addEventListener` and Wails `EventsOn`)
- Fixed title bar drag swallowing button clicks by guarding `StartDrag()` with `e.target.closest('button')`

## Install

Download `key-stats.exe` and run. Data is stored at `%APPDATA%/key-stats/`.

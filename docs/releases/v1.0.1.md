- Added `-H windowsgui` linker flag to suppress console window and restore taskbar icon
- Added `rsrc` step in build pipeline for proper Windows icon resource embedding
- Added `-trimpath` for smaller release binaries
- Fixed 11 Svelte a11y compiler warnings (keyboard handlers, ARIA roles, `svelte-ignore` for unavoidable cases)
- Updated GitHub Actions release workflow to match local build pipeline
